# Covid-19-detecttion-CNN
The world is facing a pandemic due to the coronavirus disease 2019 (COVID-19), named as per the World Health Organization. COVID-19 is caused by the virus called severe acute respiratory syndrome coronavirus 2 (SARS-CoV-2)

## At first two different models have been trained and fine tuned and then an ensemble of both of them is used to detect covid-19 with an improved accuracy of around 98.23 %
### Models used are-
* VGG-16
* RESNET-50
* ENSEMBLED MODEL
